## Featured Project — PAW (Physical Activity Website)

> Production-grade ASP.NET Core system integrating wearable fitness APIs into a cloud dashboard.

**What it does:** Ingests real-time heart rate, workout, and GPS data from Strava and Polar devices via OAuth 2.0 and HMAC-SHA256 validated webhooks, processes it through a layered .NET 8 architecture, and surfaces it through a structured REST API.

| Layer | Technology |
|---|---|
| API | ASP.NET Core, REST, HMAC-SHA256 webhook validation |
| Auth | Azure Entra SAML SSO, OAuth 2.0 |
| Infrastructure | Docker, MySQL, Azure |
| External Integrations | Strava API, Polar AccessLink API |
| Testing | xUnit — 5-category test suite, 80–90%+ coverage targets |

**Architecture:** 7-layer solution — `Paw.Api`, `Paw.Core`, `Paw.Infrastructure`, `Paw.Polar`, `Paw.Test`, `Paw.Web`, `Paw.Worker`

> *University research poster: Smartwatch → Cloud → Dashboard data pipelines & heart rate zone grading*
